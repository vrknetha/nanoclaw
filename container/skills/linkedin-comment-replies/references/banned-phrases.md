# Banned Phrases

Never use these in comment replies.

## Openers (Kill These)

- "Great point!"
- "Great observation!"
- "Absolutely!"
- "I couldn't agree more"
- "This is so true"
- "Love this!"
- "Thanks for sharing!"
- "This resonates"
- "Well said!"
- "Spot on!"
- "Nailed it!"
- "100% agree"
- "This is gold"

## Emphasis Crutches

- "This is key"
- "This is the real unlock"
- "Game-changer"
- "So important"
- "Let that sink in"
- "This matters because"
- "Here's the thing"
- "The truth is"

## AI Vocabulary (Replace)

| Never Use | Use Instead |
|-----------|-------------|
| leverage | use |
| unlock | find, get |
| insightful | smart, good |
| thought-provoking | interesting |
| compelling | good, strong |
| resonate | makes sense, agree |
| absolutely | yes, yep |
| definitely | yes |
| fascinating | interesting |
| remarkable | good, impressive |
| incredibly | very, really |
| fundamentally | basically |
| essentially | basically |
| significantly | a lot |
| paradigm | approach, way |
| synergy | working together |
| optimize | improve |
| facilitate | help |
| utilize | use |
| implement | do, build |
| comprehensive | full, complete |

## Bottleneck / Real Bottleneck
- "bottleneck" (any form — real bottleneck, the bottleneck, acceptance is the bottleneck)
- "real bottleneck"

## "Real X" Phrases (All Banned)
- "the real test"
- "the real question"
- "the real challenge"
- "the real issue"
- "the real problem"
- "the real work"
- "the real value"
- "the real risk"
- "the real insight"
- "the real difference"
- Any "real [noun]" framing — just say the thing directly

- "when it comes to"
- "in terms of"
- "from my perspective"
- "I would argue that"
- "it's worth noting"
- "interestingly enough"
- "to be honest"
- "if I'm being honest"
- "the reality is"
- "moving forward"
- "deep dive"
- "double down"

- "unpack this"

## AI Slop / Reply-Bot Patterns (from steipete blocklist)

These get you flagged as AI slop on X and LinkedIn. Never use in posts or comments.

- "the real unlock"
- "the bigger concern is"
- "the part nobody mentions is"
- "fantastic pattern"
- "proper agent infra"
- "can provide invaluable insight"
- generic abstraction soup about "agents, workflows, reasoning, interoperability" with no specifics
- fake sage tone with no concrete details
- templated agreement or praise that could fit under any post
- same upbeat, shallow reply shape repeated across threads
- canned "deep insight" that says nothing specific

## Performative Phrases

- "I completely agree with your perspective"
- "This perfectly encapsulates"
- "You've hit the nail on the head"
- "This is exactly what I've been thinking"
- "I've been saying this for years"
- "More people need to understand this"
- "This needs to be said more often"

## Over-Thanking

- "Thank you so much for sharing this"
- "Really appreciate you posting this"
- "Thanks for the wonderful insight"
- "Grateful for this perspective"

## Simple Replacements

| Instead of | Just say |
|------------|----------|
| "I completely agree" | "Yep" / "Exactly" / "Same" |
| "That's a great point" | "True" / "Right" |
| "Thank you for sharing" | "Thanks!" |
| "This is very insightful" | "Smart" / "Good point" |
| "I appreciate your perspective" | "Makes sense" |
| "Absolutely correct" | "Yep" |
| "Couldn't have said it better" | "Exactly" |
