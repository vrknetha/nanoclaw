# Banned Phrases

Never use these in video narration. They trigger "AI content" detectors or sound unnatural when spoken.

## Structural Clichés

| Banned | Why | Alternative |
|--------|-----|-------------|
| "Here's the thing..." | Overused opener | Just state the thing |
| "Here's the truth nobody talks about" | Clickbait | State the truth directly |
| "Let me explain..." | Meta, unnecessary | Just explain |
| "In this video, I'll show you..." | Meta-commentary | Start with the content |
| "But first..." | YouTube-ism | Don't do this |
| "Before we dive in..." | Overused | Skip it |
| "Let's unpack this" | Consultant-speak | Just explain |
| "Without further ado" | Filler | Delete |

## AI-Pattern Phrases

| Banned | Why |
|--------|-----|
| "Game-changer" | AI cliché |
| "Paradigm shift" | Corporate AI |
| "Leverage" (as verb) | Corporate AI |
| "Unlock" (potential, value, etc.) | AI marketing |
| "Revolutionary" | Hyperbole |
| "Groundbreaking" | Hyperbole |
| "Transformative" | Overused |
| "Supercharge" | Marketing speak |
| "Skyrocket" | Marketing speak |
| "Deep dive" | Consultant cliché |

## Weak Openers

| Banned | Why | Alternative |
|--------|-----|-------------|
| "So..." | Filler | Delete |
| "Well..." | Hesitation | Delete |
| "Actually..." | Filler | Delete |
| "Basically..." | Condescending | Delete |
| "Essentially..." | Filler | Delete |
| "Obviously..." | If obvious, don't say it | Delete |

## Accusatory Language

| Banned | Why | Alternative |
|--------|-----|-------------|
| "Your code is wrong" | Accusatory | "The code has issues" |
| "Your prompt was wrong" | Accusatory | "The prompt was wrong" |
| "You're doing it wrong" | Confrontational | "There's a better way" |
| "You should..." | Preachy | "Consider..." or state fact |

## Filler and Hedges

| Banned | Why |
|--------|-----|
| "Kind of" | Weakens statement |
| "Sort of" | Weakens statement |
| "A little bit" | Unnecessary |
| "Just" (minimizing) | Weakens impact |
| "Really" (as filler) | Often unnecessary |
| "Very" | Find a stronger word |
| "Quite" | British hedge |
| "Fairly" | Weak |

## Rhetorical Questions

| Banned | Why | Alternative |
|--------|-----|-------------|
| "Ever wondered why...?" | Cliché | State the why |
| "What if I told you...?" | Matrix meme, overused | Just tell them |
| "You know what the problem is?" | Patronizing | State the problem |
| "Want to know a secret?" | Clickbait | Share the insight |

## Overused Transitions

| Banned | Alternative |
|--------|-------------|
| "But here's the thing" | Use a pause, then state it |
| "The thing is" | Delete, state directly |
| "At the end of the day" | Delete, state directly |
| "When all is said and done" | Delete |
| "Long story short" | Delete |

## LinkedIn-Specific

These are especially cringe on LinkedIn:

| Banned | Why |
|--------|-----|
| "Agree?" (at end) | Engagement bait |
| "Thoughts?" | Low-effort CTA |
| "I'm humbled to announce" | False modesty |
| "Excited to share" | Overused |
| "Thrilled to" | Overused |
| "Delighted to" | Overused |

## Validation

Before finalizing a script, search for these patterns:
1. Open script in editor
2. Search (Cmd/Ctrl+F) for each banned phrase
3. Replace or delete each occurrence
4. Read aloud to catch patterns that "sound like AI"
